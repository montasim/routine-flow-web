The base surface for every panel — metric tiles, occurrence lists, settings groups.

```jsx
<Card padding="md">…</Card>
<Card interactive elevation="sm" onClick={open}>Tap to edit</Card>
```

Default is a flat hairline-bordered panel (`elevation="hairline"`). Raise with `sm`/`md`/`lg`. `interactive` adds the 1px hover lift. 14px radius across the system.
